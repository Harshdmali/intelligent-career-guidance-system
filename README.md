# 🎯 Intelligent Career Guidance System

An AI-powered web application that helps students and individuals discover the most suitable career path based on their interests, skills, and personality.

---

## 🚀 Features

✨ **Career Prediction Test**

* Users can take an interactive quiz/test
* System analyzes responses and suggests best career options

📊 **Career Roadmap**

* Step-by-step guidance for selected career
* Includes skills, courses, and future scope

🤖 **Career Guidance Chatbot**

* AI-based chatbot for instant career advice
* Helps users with queries related to career decisions

👤 **User Authentication**

* Secure login and registration system

📁 **Admin Panel (Optional)**

* Manage users and career data

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server:** XAMPP (Apache)

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/INTELLIGENT-CAREER-GUIDANCE-SYSTEM.git
```

---

### 2️⃣ Move to XAMPP htdocs

```
C:\xampp\htdocs\career_project
```

---

### 3️⃣ Start XAMPP

* Start **Apache**
* Start **MySQL**

---

### 4️⃣ Setup Database

* Open **phpMyAdmin**
* Create a new database (e.g., `career_db`)
* Import the provided `.sql` file

---

### 5️⃣ Run the Project

Open browser:

```
http://localhost/career_project
```
